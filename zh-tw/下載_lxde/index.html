下載 LXDE

試用 LXDE 而不更動您的系統

下載 Live CD 光碟映像檔

另有：AMD64 版本

下載並燒錄 LXDE ISO 映像檔，之後您不需要變更到您的系統即可試用 LXDE。

您需要以下配備才能使用此映像檔：
  • 一張空的光碟片與一台燒錄機（除非您是裝在虛擬機器中）
  • Intel 或是 AMD 等相容機器
  • 128M 以上的 RAM
  • 系統需求 »

LXDE 原始碼
LXDE 現在使用 Git 原始碼管理庫，我們將不同的 LXDE 組件放置在不同的源碼庫中。
LXDE 組件
Git 瀏覽: http://lxde.git.sourceforge.net/git/gitweb-index.cgi
如果您想要取得最新原始碼，請使用下列指令： 
git clone git://lxde.git.sourceforge.net/gitroot/lxde/[源碼庫名稱]
如果您是 LXDE 的開發者，請改用下列指令： 
git clone ssh://[您的_SOURCEFORGE_帳號]@lxde.git.sourceforge.net/gitroot/lxde/[源碼庫名稱]
PCManFM/libfm
Git 瀏覽: http://pcmanfm.git.sourceforge.net/git/gitweb-index.cgi
如果您想要取得最新原始碼，請使用下列指令： 
git clone git://pcmanfm.git.sourceforge.net/gitroot/pcmanfm/[源碼庫名稱]
如果您是 PCManFM/libfm 的開發者，請改用下列指令： 
git clone ssh://[您的_SOURCEFORGE_帳號]@pcmanfm.git.sourceforge.net/gitroot/pcmanfm/[源碼庫名稱]
