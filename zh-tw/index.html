LXDE

LXDE 的全名是 「Lightweight X11 Desktop Environment - 輕量級 X11 桌面環境」，它是由國際社群開發者共同維護，同時具備高效能以及節能兩大特點之桌面環境。它具備優美的介面，支援多國語言，標準鍵盤快捷鍵以及其他諸如分頁瀏覽檔案等功能。LXDE 使用少量的 CPU 以及記憶體資源，而且特別適合使用於 Netbook （小筆電），MID（行動上網裝置）以及比較老舊的電腦。LXDE 可以安裝在如 Ubuntu 或是 Debian 的 Linux 發行版上，而原本裝在上面的程式也可以在 LXDE 上正常運作。LXDE 原始碼以 GNU GPL（通用公共授權） 或是 LGPL（寬鬆通用公共授權）釋出。
