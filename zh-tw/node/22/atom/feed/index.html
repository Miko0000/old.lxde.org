LXDE 的全名是 「Lightweight X11 Desktop Environment - 輕量級 X11 桌面環境」，它是由國際社群開發者共同維護，同時具備高效能以及節能兩大特點之桌面環境。它具備優美的介面，支援多國語言，標準鍵盤快捷鍵以及其他諸如分頁瀏覽檔案等功能。LXDE 使用少量的 CPU 以及記憶體資源，而且特別適合使用於 Netbook （小筆電），MID（行動上網裝置）以及比較老舊的電腦。LXDE 可以安裝在如 Ubuntu 或是 Debian 的 Linux 發行版上，而原本裝在上面的程式也可以在 LXDE 上正常運作。LXDE 原始碼以 GNU GPL（通用公共授權） 或是 LGPL（寬鬆通用公共授權）釋出。

LXDE 是套使用 GPL 授權的開放原始碼桌面環境，可以使用在 Unix 或是 POSIX 相容的環境中（如 Linux）。其全名是「Lightweight X11 Desktop Environment - 輕量級 X11 桌面環境」。

LXDE 是以在低規格上的電腦運作良好的概念設計，特別適合在最近流行的小筆電（Netbook）以及其他小型電腦上使用，它是個節能快速的桌面環境。LXDE 可以使用在不同的 Linux 發行版本中（例如 Ubuntu 或是 Debian），並且可以為其發行版中多如繁星的應用程式提供桌面環境。LXDE 支援各種處理器架構，包含 Intel 與 MIPS，並且 LXDE 團隊已經計畫將桌面環境移植到如 ARM（常見於嵌入式系統，如手機）等其他硬體架構。

LXDE 計畫在 2006 年開始，當時台灣知名開發者洪任諭（PCMan）推出了第一套元件：PCMan 檔案管理員。現在 LXDE 團隊已經成為國際社群，團隊中有來自世界各地的開發者、設計者與貢獻者。

特點

輕巧
它只需要少少的 CPU 資源，而且當記憶體容量充足時表現特別出色。
快速
它甚至可以執行在 1999 年出產的老舊電腦上，而且不需要 3D 加速輔助。
省能源
它比其他常見的系統需要較少的資源執行相同的工作。
簡樸美
藉由 GTK+ 2，它擁有美觀，支援國際化的使用者介面。
使用簡單
它提供使用者選擇如 EeePC 般的應用程式桌面選單，或是如微軟 Windows 般的應用程式列表。
可自訂性
您可以輕易自訂 LXDE 的外觀。
其他功能
它提供如分頁檔案瀏覽或是執行對話框選單等功能，新程式在安裝後圖示即立即出現在桌面選單上。
組件間無相依性
每一個組件可以分開使用，並不會相依到其他組件。您可以在類 Unix 系統中選擇使用一部分的 LXDE 組件。
相容標準
它相容於 freedesktop.org 所訂定的標準。

元件

PCManFM
超輕巧，擁有分頁的檔案管理器。
LXPanel
可以使用圖形介面設定的桌面面板，提供您最需要的功能。
LXSession
標準相容的 X11 作業階段管理器，支援透過 HAL 與 GDM 關機、重新開機與暫停（LXSession Lite 是個去掉 X11 作業階段支援的精簡版本，而且更加穩定）
LXAppearance
LXAppearance 是一套新的多功能 GTK+ 佈景切換器，可以切換 GTK+ 佈景、圖示以及程式使用的字型。
LXLauncher
LXLauncher 是個桌面上的程式清單。
Openbox
輕量，相容標準，而且具備高度可調整性的視窗管理器（被 LXDΕ 採用，我們建議將此作為預設的視窗管理器，但是您也可以將 Openbox 換成其他視窗管理器如 IceWM、Fluxbox、Metacity‥等等）。
GPicView
簡潔、快速、輕量的圖片檢視器，可以立即啟動。
Leafpad
簡單輕量的文字編輯器（我們建議將此作為預設的文字編輯器）。
LXDE Common
整合 LXDE 不同元件的設定值，管理系統動作並整合顯示樣式。
LXTerminal
不相依桌面環境的 VTE-based 終端機模擬器，多個終端機視窗共用一個執行程序節省系統資源。
XArchiver
輕量、快速、不相依桌面環境的壓縮檔管理器。（我們建議您用作預設的壓縮檔管理器。）
LXRandR
螢幕設定工具，可以設定多螢幕。
LXNM
輕量級網路連線管理工具，可調校無線網路連線（僅限 Linux）。
GtkNetCat
Netcat 的圖形介面前端。Netcat 是一個讀寫 TCP 或 UDP 連線的網路工具。

系統需求

LXDE 已經在以下硬體配備測試，而且運作良好。

CPU RAM 硬碟 速度
Pentium II 266 MHz 192 MB 5400 rpm 中等 - 快速
VIA 400 MHz 256 MB 5400 rpm 中等 - 快速
Pentium III 600 MHz 512 MB 4500 rpm 快速
使用 AMD Athlon 1600+ 的 QEMU 虛擬機器 128 MB QEMU 磁碟映像檔 快速
AMD Athlon 1600+ (1.4 GHz) 512 MB 7200 rpm 非常快速

建議硬體配備

  • LXDE 的硬體配備需求接近 Microsoft Windows 98（可能再高一點），舊的 Pentium II CPU 即可執行
  • 在 X11 和 LXDE 啟動之後，i386 機器上的總記憶體使用量大約是 45 MB（因為不同的系統設定可能有所誤差）。
  • 雖然 LXDE 本身不需要很好的硬體，但是在 X 底下的其他程式卻很需要。例如 Firefox 和 OpenOffice.org 就非常消耗記憶體，所以我們建議您使用多於 128 MB 的 RAM。
