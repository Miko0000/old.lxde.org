  • A blank CD/USB-pendrive and the ability to “burn” blank CDs
    (unless dealing with a Virtual Machine)
  • An Intel or AMD compatible computer
  • 128M of RAM or more

2. LXDE and LXQt Source Code

The components are available from a number of git repositories, mostly one per component to make it easier to pick and choose. Both LXDE and LXQt based components are living next to each other. Please check the path carefully.

Source Components

  • Git Browse: http://git.lxde.org/
  • If you want to get the fresh-new source code, please use the following Git command: 
    git clone git clone git://git.lxde.org/git/ + project path.
