2. Código-fonte do LXDE

Você pode acessar o código fonte do LXDE através do SVN. A navegação na árvore Subversion lhe dá uma grande visão do status atual do código deste projeto. Você pode também ver o histórico completo de qualquer arquivo no repositório.

SVN do Projeto LXDE: http://sourceforge.net/svn/?group_id=180858

Navegando o Subversion do LXDE: http://lxde.svn.sourceforge.net/viewvc/lxde/
O repositório subversion do LXDE pode ser recuperado através do seguinte comando SVN: 
		svn co https://lxde.svn.sourceforge.net/svnroot/lxde lxde

SVN do PCManFM para o LXDE: http://sourceforge.net/svn/?group_id=156956

Navegando o Subversion do PCManFM: http://pcmanfm.svn.sourceforge.net/viewvc/pcmanfm/
O repositório subversion do PCManFM pode ser recuperado através do seguinte comando SVN: 

		svn co https://pcmanfm.svn.sourceforge.net/svnroot/pcmanfm pcmanfm
