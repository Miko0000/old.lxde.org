2. LXDE Código Fuente (Source Code)

Puedes acceder al código fuente de LXDE via SVN. Revisando el árbol de Subversion le da una visión general sobre el estado actual del código del proyecto. También se puede observar un historial completo de cualquier archivo en el repositorio.

LXDE Proyecto SVN: http://sourceforge.net/svn/?group_id=180858

LXDE Subversion: http://lxde.svn.sourceforge.net/viewvc/lxde/
El repositorio de Subversion de LXDE puede ser visto a través de SVN con el siguiente comando: 
		svn co https://lxde.svn.sourceforge.net/svnroot/lxde lxde

LXDE PCManFM SVN: http://sourceforge.net/svn/?group_id=156956

PCManFM Subversion: http://pcmanfm.svn.sourceforge.net/viewvc/pcmanfm/
El repositorio de Subversion de PCManFM puede ser visto a través de SVN con el siguiente comando: 

		svn co https://pcmanfm.svn.sourceforge.net/svnroot/pcmanfm pcmanfm
